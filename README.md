# Ruby Flappy Bird

A terminal-based implementation of Flappy Bird using Ruby and ASCII graphics.

## Setup

1. Install dependencies:
```bash
bundle install
```

2. Run tests:
```bash
bundle exec rspec
```

3. Run the game:
```bash
ruby lib/game.rb
```

## Game Controls

- Press SPACE to make the bird jump
- Press Q to quit the game

## Project Structure

- `lib/` - Main game code
- `spec/` - Test files
- `Gemfile` - Project dependencies
